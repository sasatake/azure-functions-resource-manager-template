# Azure Resource Template Sample

## Target Resource

#### Initialize{#init}

- Resource Group
- Storage Account
  - Blob Container

## Required

- [Azure CLI](https://docs.microsoft.com/ja-jp/cli/azure/?view=azure-cli-latest)

## Usage

- [Initialize] Resource Group

```shell
$ sh deploy.sh
```

- Delete Resouce Group

```shell
$ sh deploy.sh clear
```
